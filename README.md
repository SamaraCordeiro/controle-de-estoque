# Controle de Estoque

Projeto para gestão de estoque, desenvolvido durante a 3ª semana do processo Trainee da iJunior.

# Configuração do Banco de Dados

1. Crie manualmente a pasta `db`
2. Crie o arquivo `estoque.csv` com o cabeçalho na ordem: nome,peso,valor,quantidade


Como este é meu primeiro contato com TypeScript, Node.js e o conceito de versionamento com .gitignore, entendo que essa pode não ser a melhor abordagem, mas foi a solução que pensei ser mais prática nesse caso. Estou aberta a feedbacks e sugestões, muito obrigada :)
